# 📊 Excel 图片处理工具（Streamlit 网页应用）

> 上传包含图片链接的 Excel 文件，自动下载图片并嵌入表格，生成可下载的 Excel 文件。

---

## 🌟 功能

* 自动识别 Excel 中的图片链接
* 将图片插入 Excel 对应单元格（大小自适应单元格）
* 支持下载处理后的 Excel 文件
* 网页端交互操作，无需安装复杂环境

---

---

## 🔒 隐私声明

用户上传的 Excel 文件 仅在当前应用会话中处理，不会被保存或上传到任何外部服务器。

应用会生成新的 Excel 文件供用户下载，原始文件和生成文件不会被存储或共享。

应用不收集任何个人信息、图片链接或表格数据。

---

## 📌 使用建议

* 尽量保证 Excel 图片链接有效，否则下载会失败
* 对于大量图片处理较慢，请耐心等待进度条完成
* Streamlit Cloud 部署可直接访问网页端，无需安装
